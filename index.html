Java Project

Cinema Ticket Booking System

This is a group project developed to book and manage cinema tickets.

  • Customer/manager can register on the system.
  • Allow customer/manager to create and manage profile.
  • Customer can view & book tickets for a variety of films.
  • Manager can add, update and delete films from the schedule.
  • Written in Java & Java Swing.
  • Implements model, view, controller architecture.
  • All information stored on an SQL database.
  • Uses Java Database Conectivity (JDBC).
